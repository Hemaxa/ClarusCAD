# ClarusCAD

**ClarusCAD** — 2D CAD-приложение для создания и редактирования чертежей, разработанное на C++ с использованием Qt 6.

## Возможности

### Примитивы
- 📏 **Отрезок** — построение по двум точкам
- ⭕ **Окружность** — по центру/радиусу, центру/диаметру, 2 или 3 точкам
- 📐 **Прямоугольник** — по диагонали, центру или углу, с поддержкой скругления
- 🌙 **Дуга** — по 3 точкам или центру и углам
- 🔵 **Эллипс** — по центру и полуосям
- ⬡ **Многоугольник** — вписанный или описанный с произвольным числом сторон
- 〰️ **Сплайн** — Catmull-Rom сплайн по контрольным точкам

### Привязки
- Сетка, конечные точки, середины, центры
- Квадранты окружностей и эллипсов
- Пересечения примитивов
- Перпендикуляр к отрезкам, граням прямоугольников и многоугольников
- Касательная к окружностям, дугам и эллипсам

### Стили линий
- Сплошная основная, тонкая, волнистая
- Штриховая, штрих-пунктирная
- Линия с изломами (зигзаг)
- Конструктор пользовательских стилей

### Интерфейс
- Темы оформления: ClarusCAD, Dark, Light, Hello Kitty
- Панели: инструменты, свойства, объекты сцены, настройки
- Консоль для текстового ввода команд
- Масштабирование, панорамирование, поворот вида

## Требования

- **Qt** 6.5 или выше
- **CMake** 3.16+
- **Компилятор** с поддержкой C++17 (Clang, GCC, MSVC)

## Сборка

```bash
git clone https://github.com/Hemaxa/ClarusCAD.git
cd ClarusCAD
mkdir build && cd build
cmake ..
cmake --build build -j$(sysctl -n hw.ncpu) 
.build/ClarusCAD.app/Contents/MacOS/ClarusCAD
```

## Структура проекта

```
src/
├── controller/         # Инструменты создания примитивов
│   └── tools/          # Segment, Circle, Arc, etc.
├── model/              # Модели данных
│   └── primitives/     # Segment, Circle, Rectangle, etc.
└── view/               # UI-компоненты
    ├── drawing/        # Отрисовщики примитивов
    ├── managers/       # Менеджеры (темы, привязки, настройки)
    ├── panels/         # Панели интерфейса
    ├── properties/     # Виджеты свойств примитивов
    └── widgets/        # Компоненты UI
```
