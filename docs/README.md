# ClarusCAD

**ClarusCAD** — настольная 2D CAD-система для построения, редактирования и оформления чертежей. Проект разрабатывается на **C++** с использованием **Qt 6** и ориентирован на работу с базовой геометрией, размерными инструментами, пользовательскими стилями линий и настраиваемым интерфейсом.

Программа сочетает интерактивное построение на рабочем поле, панель свойств для редактирования выбранных объектов, систему геометрических привязок и набор размерных инструментов, которые позволяют не только отображать размеры, но и в ряде случаев перестраивать связанную геометрию через изменение числового значения.

## Назначение программы

ClarusCAD предназначен для создания и редактирования двумерных чертежей и эскизов. Программа подходит для:

- построения базовой геометрии;
- работы с окружностями, дугами, прямоугольниками, эллипсами, многоугольниками и сплайнами;
- постановки линейных, радиальных, диаметральных и угловых размеров;
- настройки визуального оформления линий и размеров;
- подготовки DXF-совместимого геометрического материала.

## Текущее состояние возможностей

На текущем этапе ClarusCAD уже включает достаточно широкий набор функций для повседневной 2D-работы.

### Геометрические примитивы

Поддерживаются следующие типы объектов:

- отрезок;
- окружность с несколькими режимами построения:
  по центру и радиусу, по центру и диаметру, по двум точкам, по трем точкам;
- прямоугольник:
  по двум точкам, по центру и размерам, по углу и размерам;
- дуга:
  по трем точкам, по центру, началу и концу;
- эллипс;
- правильный многоугольник;
- сплайн.

### Размерные инструменты

В программе реализованы:

- линейный размер;
- горизонтальный размер;
- вертикальный размер;
- радиальный размер;
- диаметральный размер;
- угловой размер.

Размеры поддерживают:

- пользовательские стили линий и стрелок;
- выбор типа размерной и выносной линии;
- настройку цвета текста, размерной и выносной линии;
- выбор типа стрелок;
- глобальные настройки текстов и размеров через окно настроек;
- редактирование значения размера с перестроением связанной геометрии;
- двойной клик по размеру для ввода нового значения;
- интерактивное редактирование через ползунки на рабочем поле;
- ассоциативное поведение для части примитивов и сценариев привязки.

### Геометрические привязки

Система привязок включает:

- привязку к сетке;
- конечные точки;
- середины отрезков и сторон;
- центры;
- пересечения;
- квадранты;
- ближайшую точку на примитиве;
- перпендикуляр;
- касательную.

Это позволяет строить и редактировать объекты с высокой точностью без ручного ввода координат в большинстве стандартных сценариев.

### Редактирование объектов

В программе доступны:

- выделение одиночных объектов;
- выделение рамкой;
- редактирование параметров через панель свойств;
- изменение слоя;
- изменение цвета;
- изменение типа линии;
- удаление объектов;
- перемещение;
- редактирование размеров через значения и интерактивные ручки.

### Стили линий и оформление

ClarusCAD поддерживает:

- набор стандартных типов линий;
- тонкие и основные сплошные линии;
- штриховые и штрихпунктирные линии;
- волнистые линии;
- линии с изломами;
- пользовательские стили линий через встроенный конструктор;
- настройку глобального стиля размеров;
- выбор тем оформления интерфейса.

## Интерфейс

Интерфейс программы построен по панельному принципу и включает:

- панель инструментов;
- рабочую область с системой координат и сеткой;
- панель свойств объекта;
- панель объектов сцены;
- панель навигации;
- панель настроек сцены;
- консоль команд.

Поддерживаются масштабирование, панорамирование, поворот вида и работа с разными темами интерфейса.

## Импорт и экспорт

В проекте реализована работа с форматом **DXF**:

- импорт DXF;
- экспорт DXF.

Это позволяет использовать ClarusCAD в связке с другими CAD-инструментами и проверять совместимость геометрии в распространенном обменном формате.

## Технический стек

Проект основан на следующих технологиях:

- **C++17**;
- **Qt 6**;
- **CMake**.

## Требования

Для сборки и запуска проекта необходимы:

- Qt 6.5 или выше;
- CMake 3.16 или выше;
- компилятор с поддержкой C++17:
  Clang, GCC или MSVC.

## Сборка проекта

Пример типовой сборки:

```bash
git clone https://github.com/Hemaxa/ClarusCAD.git
cd ClarusCAD
mkdir build
cd build
cmake ..
cmake --build . -j$(sysctl -n hw.ncpu)
./ClarusCAD.app/Contents/MacOS/ClarusCAD
```

Если сборка выполняется в другой среде или на другой платформе, путь к исполняемому файлу может отличаться.

## Структура проекта

```text
src/
├── controller/         # Инструменты построения, редактирования и логика пользовательского ввода
│   └── tools/
├── model/              # Модели сцены, примитивов и размерных объектов
│   └── primitives/
├── io/                 # Импорт и экспорт DXF
└── view/               # Интерфейс приложения
    ├── drawing/        # Отрисовщики примитивов
    ├── managers/       # Менеджеры тем, настроек, привязок и стилей
    ├── panels/         # Панели интерфейса
    ├── properties/     # Виджеты свойств объектов
    └── widgets/        # Вспомогательные UI-компоненты
```

## Архитектурные особенности

Проект логически разделен на три основных слоя:

- **model** — хранение и вычисление геометрии;
- **view** — интерфейс, отрисовка, панели и визуальное взаимодействие;
- **controller** — инструменты построения и редактирования, связывающие ввод пользователя с моделью.

Такое разделение упрощает развитие проекта, добавление новых примитивов, инструментов и режимов редактирования.

## Что уже особенно важно в текущей версии

Наиболее сильные стороны текущего состояния ClarusCAD:

- гибкая система построения примитивов;

- размерные инструменты с интерактивным редактированием;
- пользовательские стили линий;
- DXF-импорт и экспорт;
- настраиваемый интерфейс на Qt.

- углубление параметрического редактирования;
